Nouveaux éléments en HTML5

Contenu inclus :

Intégrer une vidéo sans Plug-ins externes : 

 <video controls src=''votrevideo.mp4''></video>




Intégrer un fichier audio :

<audio src="dancehall.wav" autoplay></audio>

La balise source est utilisée pour définir différents formats alternatifs d'un média au sein des éléments audio ou video pour permettre un support plus large du média par les différents navigateurs (ceux-ci ne supportant pas les mêmes formats). 



<audio controls>
<source 	src="lecon3.ogg">
<source 	src="lecon3.mp3">
<source  src="lecon3.wav">
</audio>





La balise CANVAS:

L’élément canvas est une partie cruciale de HTML5 qui ne fonctionne qu’avec des navigateurs compatibles HTML5, Il faut donc avertir les utilisateurs qu’ils ont besoin d’un navigateur HTML5.L'élément canvas est un composant de HTML qui permet d'effectuer des rendus dynamiques d'images bitmap via des scripts Pour vérifier si cela fonctionne avec un navigateur : le plus simple et le plus informatif (pour l’utilisateur) est de placer un message dans le conteneur canvas. Seuls les utilisateurs ne disposant pas de navigateurs compatibles HTML5 verront le texte dans le conteneur.



<canvas id="mon_canvas" width="350" height="350">
Texte alternatif pour les navigateurs ne supportant pas Canvas.
</canvas>




La balise embed:

La balise embed définit un conteneur pour une application externe ou un contenu interactif (flash …).


<embed src="fichierFlash.swf" />
